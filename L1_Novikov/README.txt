файлик с кодом только один, но там используется GNU PLOT - штука для графической отрисовки всех графиков. 
Так что для использования кода надо его себе скачать....

на убунте:
sudo apt install g++ gnuplot

### Что программа выведет:
В консоль: информация о всех созданных файлах и что куда записано
В файлы: все данные для построения графиков
Графики: в отдельный файл .png, картиночкой
